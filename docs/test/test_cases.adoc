== Test Cases

Die Nutzer müssen über das HTW Netzwerk im Online Tool angemeldet sein und eine stabile Internetverbindung besitzen.

=== Test Case 01- Projekt erstellen


== Vorbedingung

1. Als Dozent eingeloggt
2. Öffnen des Menüpunktes 'Projektverwaltung'
3. Öffnen des Unterpunktes 'Projekt erstellen'
4. Eingabeseite zur Erstellung eines Projektes 

== Testablauf

1. Daten werden vollständig und richtig eingegeben ohne inkonsistente Daten.
2. Daten werden vollständig eingegeben, jedoch wurde die ID mehrfach versucht zu vergeben
3. Pflichtfelder werden ausgelassen

=== Test Case 02- Teams generieren

Vorbedingung

1.	Als Dozent eingeloggt
2.	2. Öffnen des Menüpunktes ‘Teamsverwaltung‘
3.	Öffnen des Unterpunktes ‘Teams generieren‘
4.	Anzeige des Eingabefeldes zur Bestimmung der Teamgröße 

Testablauf

1.	Daten sind richtig eingegeben wurden (nur Zahlen)
2.	Daten sind falsch eingegeben wurden (alles außer Zahlen)
3.	Keine Daten werden eingegeben

Eingabedaten

1.	Richtige Beispieleingabe
2.	Falsche Beispieleingabe
3.	Keine Beispieleingabe

Nachbedingungen

1. 

* Teams werden mit gewünschter Mitgliederanzahl generiert

2. 

* Fehlermeldung erscheint
* Keine Generieurung der Teams
* Eingabe mit richtigen Daten wiederholen

3.

* Fehlermeldung erscheint
* Teamsgenerieung nach Eingabe richtiger Daten
